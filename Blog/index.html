Html

¿Que es html? Html es un Lenguajes de etiquetas,todo va dentro de etiquetas (<>) . Es el encargado de la estructura de una página Web

Se divide en dos grandes grupos head y body. En el head se encuentra los archivos externos css, javascript, titulo, etiquetas meta lo que se ubica aca no se refleja en la pagina. Body van las etiquetas que se van a ver en nuestra pagina.

Dentro de head se pueden ingresar:

Title: cambia el texto en la pestaña del navegador, es importante para posicionarse en los buscadores.

Agregar una hoja esterna link rel = "stylesheet" href="mishojasdeestilos.css"

En el body va:

header : Es la cabecera puede ir el titulo de nuestra pagina

nav: barra de navegacion

aside : barra lateral

aticle : para agrupar contenido similar

figuere: insertar imagen

img

background-image: url(../img/html5.jpg);

figcaption: agrega texto al pie de la imagen

footer: pie parrafo

mark: marca texto sin importancia ni enfasis
em : Para indicar enfasis, sustituye a etiqueta i
strong: marca un texto importante
small: presentar textos legales
cite: titulos de libros, canciones pelicular etc
addres: informacion de contact,debe ir en footer
time: representa fechas y horas

CSS

CSS va a dar estilos a los documentos html

Sintaxis

selector {
propiedad: valor;
}

Es una etiqueta de bloque y la paso a linea. li{ display:inline }

aside{ float:left; }

Tipos de selectores: etiqueta, id, clase

Etiqueta

body{ propiedad:valor; background-color:#00f; }

id: un unico documento

id="cabeceraweb" en html #cabeceraweb{ background-color:red; } en css

Clase

class="css" .css{ color:gold; }

p[name="miselementos"]{ color:red; }

Pseudoclases

p:nth:child(2){ color:blue; }

Centrar elementos

#cabeceraweb{ text-align:center; centra dentro del contenedor
margin:15 px auto; lo centra en su padre
padding: 15px; 15 px por todos los costados
}

Border

#cabeceraweb{ border:3px solid #f00; tamaño tipo color
border-radius: 20px 5px 15px 40px; redondeado
box-shadow: #999 45px 5px 10px; sombra color derecha abajo difuminado
}

Fuente

h1 { font: bold 20px verdana, sans-serif;}
fuente: tipo tamaño letra;

Modelo caja-contenedor

Etiquetas block

Las etiquetas block ocupan el 100% del ancho de la pantall

Etiquetas inline

Las etiquetas inline se colocan una al costado de la otra

Margin Padding Border

Margin: Espacio de la caja desde el borde al exterior

Padding: Espacio de la caja desde el borde hacia el interior

Border: El limite de la caja
